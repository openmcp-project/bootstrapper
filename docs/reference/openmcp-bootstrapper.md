## openmcp-bootstrapper

The openMCP bootstrapper CLI

### Synopsis

The openMCP bootstrapper CLI is a command-line interface
for bootstrapping and updating openMCP landscapes.

### Options

```
  -h, --help               help for openmcp-bootstrapper
  -v, --verbosity string   Set the verbosity level (panic, fatal, error, warn, info, debug, trace) (default "info")
```
